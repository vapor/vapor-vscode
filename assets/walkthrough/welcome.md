# Homebrew

If you do not have Homebrew yet, visit [brew.sh](https://brew.sh/) for install instructions.

```sh
brew install vapor
```

# Makefile

If you want, you can build the Toolbox from source.
View the Toolbox's [releases](https://github.com/vapor/toolbox/releases) on GitHub to find the latest version.

```sh
git clone https://github.com/vapor/toolbox.git
cd toolbox
git checkout <desired version>
make install
```

For more information and installation methods, see the [Toolbox page](https://github.com/vapor/toolbox).

> **Note**
> If you only need Leaf language support, you don't need to install the Toolbox.
